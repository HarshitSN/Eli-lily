# Mason - AI Unit Testing Agent

Mason generates unit tests automatically to improve coverage and catch edge cases faster.

## What It Does

- Generates meaningful unit tests from existing source code
- Suggests edge-case scenarios and negative test paths
- Helps improve confidence before merges
- Speeds up developer feedback loops in CI workflows

## Best For

- Teams increasing automated test coverage
- PR quality improvements before review
- Faster regression safety checks
- Standardizing test quality across codebases
